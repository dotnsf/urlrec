# urlrec

## Overview

Fetch and store HTML from URL.


## Prerequisite

- Node.js & npm installed

- IBM Cloud account

- IBM Cloudant service instance


## Install

- Edit settings.js with username & password of IBM Cloudant

- `$ npm install`

- `$ node app`
